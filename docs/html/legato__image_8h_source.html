118  Summary:
119  Describes an image asset.
120 
121  header - standard asset header
122  format - the format of the image. this directly affects which decoder

124  width - the width of the image in pixels
125  height - the height of the image in pixels
126  colorMode - the color mode of the image
127  compType - the compression mode of the image
128  flags - indicates of the mask field is used
129  mask - may contain a masking color for the image. blit operations may
130  reference this value and reject image pixels that match this
131  value. This can either be a single color mode or a pointer to an
132  image map data array.
133 
134  alphaMask - pointer to an array of per-pixel alpha blending values
135 
136  palette - will contain a valid pointer to a palette asset if thie image
137  is an index map instead of a color image
138 
139  leStreamDescriptor header - standard stream header
140  format - the format of the image. this directly affects which decoder
141  is invoked when rendering the image
142  lePixelBuffer buffer - the buffer that describes this image data
143  leImageFlags flags - image descriptor flags
144 
145  union
146  {
147  leColor color; // single masking color
148  leImageMap* map; // color mask map
149  } mask; // defines mask techniques
150 
151  leImageMap* alphaMap; // placeholder for blending mask
152 
153  lePalette* palette; // lookup palette for this image
154 */
