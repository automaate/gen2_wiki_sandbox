37 
38  Summary:
39  Defines a variable heap object. A variable heap is an allocator is
40  capable of allocating blocks of a variable size. This heap is flexible
41  in capability but suffers from fragmentation and possibly slower allocation
42  times than a fixed heap. Good for larger allocation sizes.
43 
44  leBool initialized - indicates whether the heap has been properly initialized
45  size_t size - the overall size of the heap
46  size_t used - the amount of space the heap is currently using
47  size_t maxUsage - the highest amount of usage the heap has seen
48  void* data - pointer to the heap data
49  void* allocList - a linked list of allocation data blocks
50  void* freeList - a linked list of free data blocks
51  uint32_t allocBlockCount - the number of blocks currently allocated
52  uint32_t freeBlockCount - the number of blocks in the free list
53 */
