35  Summary:
36  Defines a variable heap object. A variable heap is an allocator is

38  in capability but suffers from fragmentation and possibly slower allocation
39  times than a fixed heap. Good for larger allocation sizes.
40 
41  leBool initialized - indicates whether the heap has been properly initialized
42  size_t size - the overall size of the heap

44  size_t maxUsage - the highest amount of usage the heap has seen
45  void* data - pointer to the heap data
46  void* allocList - a linked list of allocation data blocks
47  void* freeList - a linked list of free data blocks
