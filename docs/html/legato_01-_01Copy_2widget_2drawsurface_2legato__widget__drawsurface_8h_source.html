97 /* Structure:
98  leDrawSurfaceWidget
99 
100  Summary:
101  Implementation of a Drawsurface widget.
102 
103  Description:
104  A draw surface widget is a special widget that allows an application to
105  perform custom HAL draw calls during Legato's paint loop. To use, create
106  and add a draw surface widget to the desired place in the widget tree.
107  Then register for the callback through the API
108  'leDrawSurfaceWidget_SetDrawCallback'. This callback occurs during the
109  paint loop. The application should then be free to adjust the HAL
110  draw state and issue draw calls as desired. The HAL layer, buffer, or
111  context state must not be adjusted in any way.
112 
113  It is also important to not stall for too long during the draw callback.
114 
115  Remarks:
116  None.
117 */
