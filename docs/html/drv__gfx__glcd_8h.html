Detailed Description

GLCD driver functions and definitions.

The GLCD device driver provides a simple interface to manage the GLCD module on Microchip PIC32MZ DA microcontrollers. This file defines the interface Declarations for the GLCD driver.

This driver is configured via MHC and generated specific to the configuration and hardware architecture set in MHC.

LE GLCD Display Driver component
