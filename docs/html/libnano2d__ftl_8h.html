Detailed Description

2DGPU driver functions and definitions.

The 2DGPU device driver provides a simple interface to manage the 2DGPU module on Microchip PIC32MZ DA microcontrollers. This file defines the interface Declarations for the 2DGPU driver.

This driver is configured via MHC and generated specific to the configuration and hardware architecture set in MHC.

See also
LE 2DGPU Graphics Processor component
