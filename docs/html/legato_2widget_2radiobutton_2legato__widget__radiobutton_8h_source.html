377  leRadioButtonWidget
378 
379  Summary:
380  Implementation of a radio button widget struct
381 
382  Description:
383  A radio button is similar to a checkbox widget in that it has an on and
384  off state. It is further capable of being added to a radio button group.
385  This group provides a mutually exclusive selection capability so that
386  only one radio button may be selected at any one time.
387 
388  Remarks:
389  None.
390 */
