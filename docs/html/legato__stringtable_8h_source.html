53 /* Structure:
54  leStringTable
55 
56  Summary:
57  Describes a string table asset. There is typically only ever one of these
58  defined at any one time.
59 
60  header - standard asset header
61  languageCount - the number of languages in the string table
62  stringCount - the number of strings in the string table
63  stringIndexTable - the pointer to the string index table. the string index
64  table is a table that contains all of the unique strings
65  defined in the string table.
66  fontTable - the font table contains an array of pointers to all defined
67  font assets that the string table references
68  fontIndexTable - the font index table is a table that maps strings to font
69  indices which can then be used to get an actual font pointer
70  from the font table
71  encodingMode - indicates how strings are encoded in the stringIndexTable
72 */
